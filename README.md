# log-analyzer
The purpose of creating this file is to demonstrate how to perform log analysis using Python

This project is a simple and efficient log analysis tool written in Python.
It reads a log file, counts different log levels (INFO, DEBUG, ERROR, WARNING),
prints a summary in the terminal, extracts error logs, and saves a report as a text file.
 Features
 Reads log data from app.log
 Counts occurrences of:
INFO
DEBUG
ERROR
WARNING

Detects all log levels case-insensitively
Prints a clean summary in the terminal
 Extracts and displays only ERROR logs
 Saves the final report to report.txt
